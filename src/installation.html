Installation and Quick Start

First download latest Sleek dashboard from here. We also provide CDN link (see code below). To start using the dashboard you will need to import some files in your current project or start from scratch using our template (scroll down in this page and see Starter Template).

CSS

Copy-paste the stylesheet link into your head. Please note that you have to include required plugins stylesheet file before this (bootstrap.min.css is bundled with sleek.min.css. You can see Starter Template to get idea)

<!-- CSS Files for required plugins -- >
<!-- --- -- >
<!-- --- -- >

<link href="assets/css/sleek.min.css" rel="stylesheet">

<!-- or from CDN  --
<link href=" https://unpkg.com/sleek-dashboard@1.0.0-beta.1/dist/assets/css/sleek.min.css" rel="stylesheet">
JS

Place the following scripts near the end of your pages, right before the closing body tag, to enable them. Please note that you have to insert another required plugins script files. jQuery must come first, then another required library, then sleek.js (popper.js, and bootstrap.js bundled with sleek.bundle.js. You can use Starter Template to get idea)

<!-- JS Files for required plugins -- >
<!-- --- -- >
<!-- --- -- >

<script src="assets/js/sleek.bundle.js" > </script >

<!-- or from CDN --
<script src=" https://unpkg.com/sleek-dashboard@1.0.0-beta.1/dist/assets/js/sleek.bundle.js" > </script >
Alternatively if you want to use sleek dashboard from npm package run npm i sleek-dashboard --save in your terminal

Starter Template

Be sure to have your pages set up with the latest design and development standards. That means using an HTML5 doctype and including a viewport meta tag for proper responsive behaviors. Put it all together and your pages should look like this:


<!DOCTYPE html>
<html lang="en" dir="ltr">

<head>
<meta charset="utf-8" />
<meta http-equiv="X-UA-Compatible" content="IE=edge" />
<meta name="viewport" content="width=device-width, initial-scale=1" />
<title>Sleek Starter - Admin Dashboard Template</title>

<!-- GOOGLE FONTS -->
<link href="https://fonts.googleapis.com/css?family=Montserrat:400,500|Poppins:400,500,600,700|Roboto:400,500"
rel="stylesheet" />
<link href="https://cdn.materialdesignicons.com/3.0.39/css/materialdesignicons.min.css" rel="stylesheet" />
